## Installation

-   Clone this repository
-   Open with text editor
-   Install all dependencies
    ```bash
    > composer install
    ```
-   Initialization

    ```bash
    > php init
    ```

-   Migrate database
    ```bash
    > php yii migrate
    ```

## Run

-   Run use local development PHP
    ```bash
    > php -S 127.0.0.1:8000 -t frontend/web
    ```
-   You can access `127.0.0.1:8000` in browser
